Got Claude Code running on GPT-5.6 Sol through CLIProxyAPI.

I made two tiny wrappers:

- `./claudex` → Azure OpenAI
- `./claudex-oai` → my OpenAI/Codex subscription

Both keep effort mode on, auto-start the proxy, and work with Claude Code tools. Azure prompt caching works too.

Nice way to keep the orange crab while choosing where the model comes from. Credit to Theo for the idea.
